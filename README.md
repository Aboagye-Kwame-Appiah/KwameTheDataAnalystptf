# Kwame Aboagye Appiah - Data Analyst Portfolio

A professional portfolio website showcasing the skills, projects, and experience of Kwame Aboagye Appiah, a QA-grounded Data Analyst specializing in Power BI, Python, SQL, and operational analytics.

## Features

- **Home Page**: Introduction and overview of services.
- **About**: Personal story, journey, and core values.
- **Skills**: Technical competencies, proficiency matrix, and certifications.
- **Projects**: Detailed case studies of analytics projects with quantified outcomes.
- **Tech Stack**: Comprehensive list of tools, workflow, GitHub stats, and learning roadmap.
- **Contact**: Contact form, availability, and quick action links.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Dynamic Content**: GitHub repository data fetched and integrated into the site.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with Google Fonts (Cormorant Garamond, Outfit)
- **Backend/Scripting**: Python (for data fetching)
- **Automation**: GitHub Actions (update-portfolio.yml)
- **Version Control**: Git

## Installation and Setup

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your-username/portfolio.git
   cd portfolio
   ```

2. **Fetch GitHub Data** (Optional, for dynamic content):
   - Ensure Python is installed.
   - Install required packages (if any, e.g., requests):
     ```bash
     pip install requests
     ```
   - Run the fetch script:
     ```bash
     python fetch_repos.py
     ```
   - This will update relevant sections with live GitHub data.

3. **Open the Website**:
   - Open `index.html` in your preferred web browser.
   - Navigate through the site using the menu.

## Usage

- Browse the portfolio to view skills, projects, and contact information.
- The site is static and can be hosted on any web server or GitHub Pages.
- For updates, modify the HTML/CSS/JS files directly or run scripts to refresh dynamic content.
